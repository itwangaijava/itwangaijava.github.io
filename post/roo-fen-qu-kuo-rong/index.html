roo分区扩容

· 2022-03-23 · # linux

U盘是这个路径
sudo fdisk /dev/sda

1、进入分区管理
输入p，看到两个分区，第一个分区不要动，我们改动第二个分区。
注意，记住/dev/mmcblk0p2的start起始块数
sudo fdisk /dev/mmcblk0
2、删除分区

输入d ， 再输入2，删除第二分区

接着输入p，确认只剩下一个分区
3、创建分区

输入n，创建一个分区
输入p，建立主分区

输入2，输入起始块数

接着回车，默认能够分配到最大

最后输入y，确认

4、保存并重启

输入w，保存更改

5、修护分区

输入
sudo resize2fs /dev/mmcblk0p2

最后再查看一下

输入df -h

输入sudo reboot，重启
